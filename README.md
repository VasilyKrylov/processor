# SPU - soft processor unit

## О проекте

Данный проект реализует виртуальную стековую машину. Проект состоит из двух частей:

- asm - ассемблер, который превращает текст программы (.asm файлы) в исполняемые файлы (.spu)
- spu - виртуальный процессор, который запускает скомпилированные программы. В него входят:
  - Стэк
  - Оперативная память
  - Регистры
 
## Сборка
```
git clone https://github.com/VasilyKrylov/processor
cd processor/asm
make
cd ../spu
make
cd ..
```

## Использование

В папке my_asm/ создайте файл [name].my_asm и запустите скрипт сборки и запуска:

```
./build_and_run.sh [name]
```
